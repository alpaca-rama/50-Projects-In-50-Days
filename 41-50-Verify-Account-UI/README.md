## 41-50-Verify-Account-UI
A project just creating the UI for the verification process of signing up / creating an account.

The project allows the user to type each number (0...9) and upon entering a number it goes to the next input.

The project allows the use of backspace to backstep a number.

The project is made with HTML, CSS and JS.
