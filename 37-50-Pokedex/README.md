## 37-50-Pokedex
A Project that pulls from a [pokemon API](https://pokeapi.co/api/v2/pokemon/) to display the first generation pokemon (151).

The amount of pokemon can be increased by changing the variable amount mentioned below in the script.js file:

```javascript
const pokemon_count = 151
```

The project is made with HTML, CSS and JS.
